The DSWA stands for "Damn Secure Web App". Inspired from the official pentesting and VAPT attack vector testing platform DVWA which stands for "Damn Vulnerable Web App".


DSWA aims to provide a full-fledged platform to web devs and security researchers , to make , try and test their own attack vectors/scripts or web apps and websites againt the well known web vulneribilities.


The DSWA provides different modules for each different attack vector/types and has security for them respectively. Click any of the links above and start experimenting your Attack scripts in this platform.


Each module provides a security level of "None" to "highly secured mode" to try out and test the effectiveness of the malicious code onto a low to very secure level
